# Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (FSE) Hands-on Repository

## Overview

Welcome to my **Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (FSE)** repository.

This repository contains all the hands-on exercises, coding assignments, and practical implementations completed throughout the Digital Nurture 5.0 Deepskilling Program. Each module focuses on a different area of full-stack development, covering Java fundamentals, design patterns, databases, Spring ecosystem, frontend development, cloud technologies, DevOps, and microservices.

The primary objective of this repository is to showcase my learning progress, practical implementations, and understanding of industry-standard technologies used in enterprise application development.
