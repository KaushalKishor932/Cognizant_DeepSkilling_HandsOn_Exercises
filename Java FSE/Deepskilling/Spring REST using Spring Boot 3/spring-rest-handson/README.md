# Spring REST Hands-on

This repository contains the practical exercises developed for **Module 7 – Spring REST with Spring Boot 3** as part of the **Cognizant Java Full Stack Engineer (FSE) Deepskilling Program**. The project showcases the implementation of RESTful APIs using the Spring Boot framework and follows a layered architecture.

---

# About the Project

The objective of this project is to build and expose REST APIs using Spring Boot. It demonstrates how controllers, services, DAOs, and model classes work together to process requests and return JSON responses.

---

# Implemented Features

## Employee REST Module

The Employee API includes:

- Employee entity
- Department entity
- Data Access Object (DAO)
- Service layer implementation
- REST controller
- Global exception handling

## Country REST Module

The Country API consists of:

- Country entity
- DAO layer
- Business service layer
- REST controller
- CRUD-style REST endpoints

# Running the Application

Follow these steps to execute the project locally:

1. Import the project into IntelliJ IDEA or another compatible Java IDE.
2. Allow Maven to download all required dependencies.
3. Start the Spring Boot application.
4. Use Postman, a browser, or any REST client to access and test the available API endpoints.
